## 💡 Problem Statement

> **Predict future per capita income trends in Canada using historical data and regression models.**  
> 📅 **Date:** July 7, 2025  
> 📂 **Dataset:** `canada_per_capita_income.csv` with `year` and `per capita income` columns.

---

## 🧠 What I Did

### 🔍 Dataset Exploration
- Loaded the dataset and viewed the first few rows.
- Explored the shape and summary statistics.
- Checked for missing values to ensure clean data.

### 📈 Visualizing Relationships
- Visualized the relationship between **year** and **income** using a scatter plot.

### 📊 Linear Regression
- Applied Linear Regression to fit a straight line to the data.
- Visualized the fitted line.
- Evaluated performance using R² score.

### 🧮 Polynomial Regression
- Used degree-2 Polynomial Regression to capture curve trends.
- Improved model accuracy.
- Plotted both the predicted curve and the original data.

## ✨ Key Concepts Highlighted

- **`pd.read_csv()`**: Load the dataset  
- **`.shape`, `.describe()`**: Basic exploration  
- **`.isna().sum()`**: Missing value check  
- **`plt.scatter()`**: Visualization  
- **`LinearRegression()`**: Model fitting  
- **`.score()`**: Accuracy with R²  
- **`PolynomialFeatures()`**: Create polynomial terms  
- **`.reshape()`**: Format X for sklearn
